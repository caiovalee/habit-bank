☁ Sincronização entre dispositivos

Use um GitHub Gist privado como nuvem pessoal. Configure uma vez por dispositivo.

  1. Crie um token: github.com/settings/tokens/new → marque o escopo gist → gere e copie
  2. Crie um Gist privado: gist.github.com → nome do arquivo cofrinho.json → conteúdo {} → clique em Create secret gist
  3. Copie o ID do gist da URL (a parte longa após seu usuário)
