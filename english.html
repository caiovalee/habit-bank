Muito frequenteVerbos irregulares no passadofind→found, build→built, break→broke...
Errado
❌ "I finded a critical bug in the checkout flow."
❌ "We builded the automation suite from scratch."
❌ "The deploy breaked the staging environment."
Correto
✅ "I found a critical bug in the checkout flow."
✅ "We built the automation suite from scratch."
✅ "The deploy broke the staging environment."
Verbos irregulares essenciais em QA:

find → found  |  build → built  |  break → broke
run → ran  |  write → wrote  |  send → sent
make → made  |  take → took  |  catch → caught
know → knew  |  go → went  |  give → gave
💡 Nunca adicione -ed a esses verbos. Se reconhece o verbo, use a forma irregular.
🔧 Daily: "I ran the regression", "We found 3 blockers", "The test broke after the merge."
Muito frequentePresent Continuous — am/is/are + -ingNão esqueça o verbo "be"
Errado
❌ "I watching a video about Knot Theory."
❌ "The team working on the fix right now."
Correto
✅ "I'm watching a video about Knot Theory."
✅ "The team is working on the fix right now."
Estrutura obrigatória: am/is/are + verbo-ing

I am (I'm) testing  |  He/She is (he's) testing  |  We/They are (we're) testing

Sem am/is/are → frase incompleta e errada.
🔧 Daily: "I'm investigating the timeout", "We're waiting for the dev fix", "The pipeline is running."
FrequentePreposições: in / on / atAmbientes, plataformas e passos
Errado vs Correto
❌ "I tested on staging environment."
✅ "I tested in the staging environment."
❌ "The bug appeared in Jira."
✅ "The bug was reported on Jira."
❌ "The flow breaks on the checkout."
✅ "The flow breaks at the checkout step."
IN → ambientes/espaços: in production, in staging, in the app, in the database

ON → plataformas: on Jira, on GitHub, on GitLab, on BrowserStack, on mobile

AT → pontos específicos: at the login page, at step 3, at checkout
💡 Mantra: Ambiente = IN. Plataforma = ON. Passo específico = AT.
FrequenteModal verbs — nunca "to" depoiscan / should / must / might
Errado
❌ "You should to run the tests before merging."
❌ "This must to be a regression."
❌ "I can to investigate this tomorrow."
Correto
✅ "You should run the tests before merging."
✅ "This must be a regression."
✅ "I can investigate this tomorrow."
Regra de ouro: Modal + verbo base — NUNCA "to"

can = consegue → "I can reproduce the bug"
should = deveria → "You should open a ticket"
must = obrigatório → "We must deploy before 5pm"
might = talvez → "This might be a cache issue"
could = poderia → "It could be a race condition"
FrequenteWill/won't + past participleFuturo passivo: "will be fixed"
Errado vs Correto
❌ "The bug won't be fix before the release."
✅ "The bug won't be fixed before the release."
❌ "The feature will be deploy on Friday."
✅ "The feature will be deployed on Friday."
Estrutura: will/won't + be + past participle

fix → fixed  |  deploy → deployed  |  test → tested
break → broken  |  build → built  |  prepare → prepared
💡 Depois de "will be" / "won't be" → sempre particípio. fix→fixed, deploy→deployed.
DicaFalse cognates — armadilhas clássicasactual, pretend, realize, sympathetic...
Armadilhas com contexto
❌ "The actual president" → actual = real/genuíno
✅ "The current president" → atual = current
❌ "The politics don't care" → politics = área/assunto
✅ "Politicians don't care" → políticos = politicians
❌ "Renew my signature" → signature = nome assinado
✅ "Renew my subscription" → assinatura de serviço
atual → current  |  simpático → nice/friendly
pretender → intend/plan  |  realizar → achieve/accomplish
polêmico → controversial  |  políticos → politicians
DicaAdjetivos — posição e adverbsOrdem e modificação correta
Errado vs Correto
❌ "The state brazilian needs reform."
✅ "The Brazilian state needs reform."
❌ "This is extreme important for the release."
✅ "This is extremely important for the release."
Posição: adjetivo ANTES do substantivo em inglês.
❌ "a bug critical" → ✅ "a critical bug"
❌ "an environment unstable" → ✅ "an unstable environment"

Adverb: para modificar adjetivo, adicione -ly:
extreme → extremely  |  incredible → incredibly  |  total → totally
